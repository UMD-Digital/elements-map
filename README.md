# &lt;umd-map&gt; element

## Installation

```
$ yarn add @universityofmaryland/map
```

## Basic Usage

#### Import the map in your js if you bundle using a transpile with babel or typescript.

```js
import '@universityofmaryland/map';
```

#### HTML usage

```html
<umd-map> </umd-map>
```
